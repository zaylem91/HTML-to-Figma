How to Use

  1. Start the server using the button above
  2. Open the Figma plugin "HTML to Figma"
  3. Switch to the "URL Extraction" tab
  4. Enter any website URL
  5. Click "Extract URL" - the JSON appears automatically!
  6. Click "Convert to Figma" to create editable design
Keep this app running while using the Figma plugin for automatic extraction.
